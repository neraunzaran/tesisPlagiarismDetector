# tesisPlagiarismDetector

En este repositorio tenemos los archivos pdf de las tesis de Benigno Ramirez Choc y Consuelo Porras.
Mediante el servicio MinerU (mineru.net) se extrajo en texto plano el contenido de ambas tesis en formato Markdown.
Luego, se comparan ambas y se asignan porcentajes de similitud en parrafos completos y finalmente se crea un archivo donde viven frente a frente los parrafos que pueden ser sospechosos de plagio.
